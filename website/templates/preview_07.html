Pharma-geddon

After a few days of high-tension work in other areas of the ship, you put in a request for a more laid-back task. Helping to organize the inventory in the Sick Bay sounds like a great, mindless task that you can relax by doing.

As seems to be your usual misfortune, you arrive to yet another tableau of stress and confusion. The medical officer has just put in an order for some medical supplies but the crew are struggling with their auto-vending machine.

The steps for retrieving an item from the auto-vending storage are:

  1. Look up the four-symbol codes, in the reference sheet, for each item you need.
  2. Concatenate the four-symbol codes into one string without spaces and type it into the auto-vending machine's input field.

This seems easy enough, but someone has lost the reference sheet. Without the four-symbol code for each item, you cannot retrieve it from the auto-vending storage.

Luckily, there is a raw-data file (your puzzle input) in the computer system that stores the entire collection of medical supplies. When these items were entered into the ship's inventory, in no particular order, their four-symbol codes were concatenated into the raw data file with no separators.

You will first need to split the raw data into groups of four symbols , and then use a Counter to find the quantity associated with each four-symbol code.

The auto-vending machine is able to show you the quantity for each item, but not its four-symbol code. To get the four-symbol code for an item, you will need to match the quantity of the item shown on the auto-vending machine with the quantity for the four-symbol code you've obtained from the raw data.
